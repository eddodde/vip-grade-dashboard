# VIP 등급 수불 대시보드

VIP 등급 **수불(승급·유지·하락/유입·이탈)·전환·활성도**를 월별로 진단하는 Streamlit 대시보드.
**모든 수치는 원본 RAW 데이터에서 직접 계산**한다(사용자 수동 집계 시트는 참고만, 숫자는 재계산).

## 계산 모델 (중요)
원본 RAW는 한 이동을 '유입'(목적지 기준)과 '이탈'(출발지 기준)로 **이중 기록**한다.
→ 그대로 합치면 이동 인원이 2배가 된다(원본 `이동_방향성` 시트가 그런 케이스).
이 대시보드는 **'유입'+'유지' 행만** 써서 단일기입 전이행렬 `M[FROM(전월)→TO(당월)]` 을 만들고
모든 지표를 일관되게 파생한다(인구 보존: ΣFROM = ΣTO).

- 유지(g)=M[g,g] · 유입(g)=Σ_{f≠g}M[f,g] · 이탈(g)=Σ_{t≠g}M[g,t]
- 유지율=유지/전월유효, 전월유효=유지+이탈
- 승급/하락=등급 RANK 비교, 전환(일반→VIP)=Σ M[일반→VIP]

## 등급 체계 (상위→하위)
`SP > PT > GD > SV > BK > PP > RD`
- **VIP**: BK·SV·GD·PT·SP / **일반**: PP·RD
- **SP·PT는 연1회(매년 1/1) 고정 등급**이라 월 단위 VIP 유지율 집계에서 **제외**(= VIP core: BK·SV·GD).
  활성도 차트 등에는 선택 시 표시.

## 지표 분모 (요청 양식 정의 기준)
- **유효회원 = 유지 + 유입** (당월). 유지 구성비·유입 구성비 = ÷ 당월 유효회원 (합 100%)
- 이탈률 = 이탈 ÷ 전월 유효회원, 잔존율 = 유지 ÷ 전월 유효회원
- 유입은 승급(하위→)/하락(상위→), 이탈은 승급(→상위)/하락(→하위)으로 분해 + 출처/행선지 등급별 구성

## 섹션
핵심 요약 · **수불 한눈에(승급/유지/하락 + 등급별 유입·이탈 밸런스)** · 등급별 수불 추세 ·
VIP vs 일반 · 일반→VIP 전환 & 내부 대사 · 이동 방향성(히트맵·Sankey·등급별 수불 상세) ·
DAU/MAU 활성도 교차 · **VIP 에이징 분포**. 사이드바에 **목차(앵커 이동)** 제공.

## VIP 에이징 분포
가입후 경과월 코호트: 신규유입(0~2M)·온보딩(3~5M)·안정화(6~11M)·하락가망(12M)·장기유지(13M~)
× 유효회원/MAU/DAU (VIP 5등급). 소스 = 에이징 원본의 `2번_data` (`data/aging.csv`).
- ⚠️ 원본에 **202505월 누락**(해당 월 비어 있음).

## BK(Black) 데이터 기준 (검증 결과)
`등급수불현황…251211.xlsx`의 `1번_data_n`은 BK 유지가 16만(202507까지)인데, 이는 **휴면/제휴성
Black이 미제외**된 값(BK만 MAU율 71%→37%로 급락). 본 대시보드의 수불 템플릿(BK 3.9만)과
에이징 데이터는 **전 VIP등급 1% 이내로 정합**(BK 활성률 71%로 다른 VIP등급과 일관)이므로
템플릿 기준을 사용한다. → 등급수불·에이징 모두 동일한 '활성/제휴제외' 기준.

## 데이터 갱신 — 2가지
**수불·에이징이 한 파일**에 들어있음(시트 `등급수불` + `에이징`). 업로드/추출 모두 파일 1개면 됨.
1. **사이드바 업로드(권장)**: 통합 원본 xlsx **1개**를 올리면 git push 없이 즉시 재계산
   (앱이 `등급수불`·`에이징` 두 시트를 자동 탐지해 각각 파싱).
2. **CSV 커밋**: 아래 실행 → `data/raw_grade.csv`·`data/aging.csv` 재생성 → commit/push
   ```powershell
   ./convert.ps1 -Src "통합 원본 xlsx 경로"
   ```
   `convert.ps1` 은 Excel COM 으로 메모리에서 읽어 ASCII 헤더로 시트 탐지
   (수불 A1='CURR_STD_YM' / 에이징 c1='YM'&c7='MAU'), `data/*.csv` UTF-8(BOM) 추출.
   `-AgingSrc` 생략 시 `-Src` 와 같은 파일에서 에이징도 추출. **원본 xlsx 는 커밋 안 함**(`.gitignore`).
